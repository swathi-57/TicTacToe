Introduction
This project is a basic Tic-Tac-Toe game designed for two players. The game is built using HTML for the structure, CSS for styling, and JavaScript for the game logic. It provides a fun and interactive way to play Tic-Tac-Toe in the browser.

Features
Two-player game
Responsive design
Interactive game board
Winning and draw detection
Reset functionality
